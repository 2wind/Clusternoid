이 폴더 속 SFX 파일의 저작권

무료 sfx 사이트를 전전하다가 그냥 sfx팩을 하나 구입했습니다.
지금 이 폴더 속에 있는 sfx파일들은 다 거기서 온 것이고 앞으로도 많이 가져올 것 같습니다.
